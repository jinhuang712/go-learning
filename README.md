# Go Learning

面向 Java 资深后端工程师的 Go 学习项目，目标是能在高并发、分布式微服务场景中独立阅读、设计、评审和优化 Go 代码。

## 学习进度

- 已完成：Lesson 1 Basics、Lesson 2 Collections、Lesson 3 Structs & Interfaces
- 进行中：阶段 A（基础与心智模型）复习与巩固
- 下一步：错误处理与 context，再进入并发核心（goroutine/channel/select）

## 复习方式

- 代码回放：`go run .`
- 单元复习测试：`go test ./...`
- 并发阶段开始后建议固定加入：`go test -race ./...`
